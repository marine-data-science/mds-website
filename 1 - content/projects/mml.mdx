# Research Focus Marine Machine Learning

Machine learning is becoming increasingly important for fields like oceanography, marine ecology, and underwater technology. ML helps researchers process and interpret complex, large-scale marine data—such as satellite observations, sensor readings, and underwater imagery. At the [Institute for Visual & Analytic Computing](https://vac.uni-rostock.de/), we work closely with marine scientists to develop machine learning tools that support tasks like ecosystem monitoring, analyzing ocean physical processes, and autonomous vehicle control.
